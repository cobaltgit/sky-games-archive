# Playable Games

This is a list of playable games as well as their completability status and any bugs  

Controls for all games are as follows:

- Arrow buttons - Arrow keys/IJKL
- Select - Enter/Space
- Back up - Backspace
- Colour buttons:

    </div>
- Help - T
